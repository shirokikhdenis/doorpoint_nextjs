# Запуск Яндекс Директ и веб-аналитики

Чеклист после выкладки сайта на боевой домен.

## 1. Окружение

- [ ] В `.env` на сервере задать `NEXT_PUBLIC_SITE_URL=https://ваш-домен.ru` (HTTPS, без слэша в конце).
- [ ] Создать счётчик в [Яндекс Метрике](https://metrika.yandex.ru/) и указать `NEXT_PUBLIC_YANDEX_METRIKA_ID` в `.env`.
- [ ] Пересобрать и перезапустить приложение после смены переменных.

## 2. Яндекс Вебмастер

- [ ] Добавить сайт в [Вебмастер](https://webmaster.yandex.ru/).
- [ ] Подтвердить права (meta-тег, DNS или файл).
- [ ] Отправить sitemap: `https://ваш-домен.ru/sitemap.xml`
- [ ] Проверить `https://ваш-домен.ru/robots.txt`
- [ ] Убедиться, что главная, каталог, товары и контакты попадают в индекс.

Подробнее по Google: [docs/google-search-console.md](./google-search-console.md)

## 3. Цели в Метрике

В интерфейсе Метрики создайте цели типа «JavaScript-событие» с идентификаторами:

| ID цели | Когда срабатывает |
|---------|-------------------|
| `measure_lead` | Отправка формы «Запись на замер» |
| `cart_lead` | Отправка заявки из корзины |
| `phone_click` | Клик по номеру телефона |
| `social_click` | Клик по VK / Telegram / MAX |

## 4. Яндекс Директ

- [ ] Привязать счётчик Метрики к кампании.
- [ ] Выбрать цели для стратегии «Оптимизация конверсий» (замер и/или заявка из корзины).
- [ ] Добавить UTM в объявления: `utm_source=yandex&utm_medium=cpc&utm_campaign=...`
- [ ] Использовать релевантные посадочные:
  - витрины каталога: `/catalog?catalogPage=interior-doors`, `/catalog?catalogPage=entry-doors`
  - товар: `/product/{slug}`
  - замер: `/` (блок формы) или `/uslugi`
  - адрес и телефон: `/contact`
- [ ] Запустить тестовую кампанию с небольшим бюджетом на 1–2 недели.

## 5. Яндекс Бизнес

- [ ] Создать или обновить карточку организации.
- [ ] Сверить NAP с сайтом: телефон `+7 921 290 5999`, адрес ТЦ «Новосёл», Московский пр., 25.
- [ ] Добавить ссылку на сайт и фото салона / работ.

## 6. Проверка перед масштабированием бюджета

- [ ] Lighthouse mobile на главной, каталоге и карточке товара.
- [ ] Формы отправляются, цели фиксируются в Метрике (режим «Отладка»).
- [ ] На мобильном видны телефон и CTA в первом экране.
- [ ] Страница `/privacy` доступна, согласие под формами работает.
