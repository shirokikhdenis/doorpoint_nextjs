# Сопутствующая фурнитура на карточке ручки

Блок «Сопутствующая фурнитура» на странице товара из **Фурнитура → Ручки**.

## Когда блок показывается

Только если подбор вернул хотя бы один товар.

Триггер на бэкенде:

- корневая категория: slug `fittings`;
- подкатегория: slug `ручки`;
- на ручке заполнены **Производитель** и нужные поля для слотов (см. таблицу ниже).

## Порядок выдачи (фиксированный)

| # | Подкатегория | Доп. критерий на товаре | Совпадение с ручкой |
|---|--------------|-------------------------|---------------------|
| 1 | `защелки` | `LOCK_TYPE` = Сантехническая магнитная | производитель + **оттенок** |
| 2 | `фиксаторы` | — | производитель + артикул цвета + **розетка** |
| 3 | `петли` | `HINGE_TYPE` = Врезная универсальная | производитель + **оттенок** |
| 4 | `петли` | `HINGE_TYPE` = Бабочка | производитель + **оттенок** |
| 5 | `ограничители` | цена **≤ 1000 ₽** | производитель + **оттенок** |
| 6 | `шпингалеты` | — | производитель + **оттенок** |

Если для слота нет подходящего товара — слот пропускается, остальные показываются.

Выбор детерминированный: `ORDER BY p.id ASC`, без `RANDOM()`.

## Атрибуты

Коды `LOCK_TYPE` и `HINGE_TYPE` используются напрямую (с fallback по имени в справочнике).

Поля ручки резолвятся из `attribute_definitions` по имени:

| Поле | Имя в справочнике | Fallback-код |
|------|-------------------|--------------|
| manufacturer | Производитель | `manufacturer` |
| colorArticle | Артикул цвета фурнитуры | — |
| shade | Оттенок | `ottenok` |
| rosette | Розетка | — |

## API

```json
{
  "relatedFittings": {
    "items": [
      {
        "id": 1,
        "sku": "...",
        "name": "...",
        "price": 1190,
        "image": "...",
        "subcategory": "Защелки",
        "group": "magnetic_latch"
      }
    ]
  }
}
```
